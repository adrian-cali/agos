# AGOS APK Installation Guide (Android)

This guide is for users installing AGOS from the Google Drive folder:

- **AGOS (BETA) - Android only**

Inside that folder, users should see:

- `agos.apk`
- `ANDROID_APK_INSTALL_GUIDE.md`

## Install via Google Drive

1. Open the Google Drive folder **AGOS (BETA) - Android only**.
2. Tap `agos.apk`.
3. Tap **Download** (if prompted), then open the downloaded file.
4. If Android shows **"Install unknown apps"**:
   - Tap **Settings**.
   - Allow installs from the app you used (Google Drive / Files).
   - Go back and continue installation.
5. Tap **Install**.
6. After installation, tap **Open** to launch AGOS.

## If Installation Is Blocked

### "App not installed"

1. Remove any old AGOS version from the phone.
2. Re-download `agos.apk` from the same Drive folder.
3. Try installing again.

### Play Protect warning

1. Tap **More details**.
2. Tap **Install anyway** (for this trusted beta build).

## Important

Only install `agos.apk` from the official folder:

- **AGOS (BETA) - Android only**
